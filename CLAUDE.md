## Project Overview

Single-file HTML PWA (`llm-wiki.html`) deployed at:
**https://trashthejay.github.io/llm-wiki/**

GitHub repo: **github.com/trashthejay/llm-wiki**
Local path: `/Users/jaehochung/Desktop/Coding Project/llm-wiki/`

### What it does
- User pastes a URL or uploads a screenshot
- Gemini Flash (free API) reads and analyzes the content
- App suggests which Notion wiki page to update (or creates a new one)
- User confirms → Notion page is updated with versioned, structured knowledge
- All knowledge persists even after the source is gone; old info marked ⚠️ Outdated

---

## Architecture

```
iPhone Safari/Chrome
  → Share Sheet ("Send to Wiki" iOS Shortcut)
  → PWA opens with ?url=<shared-url>
  ↓
Gemini Flash API (generativelanguage.googleapis.com)
  → analyzes content, suggests category + topic
  ↓
Notion API (via corsproxy.io for CORS)
  → creates or updates wiki page
```

        ├── Summary section
