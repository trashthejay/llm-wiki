iPhone Setup (iOS Shortcut)
1
Open the Shortcuts app
Create a new shortcut and tap the + button.
2
Add action: "Receive URL from Share Sheet"
Search for "Receive" → select Receive Input from Share Sheet. Set type to URLs.
3
Add action: "Open URLs"
Search for "Open URLs". In the URL field, tap the variable icon and insert:
https://trashthejay.github.io/llm-wiki/?url=
Then append the Shortcut Input variable after the ?url=
4
Name it "Send to Wiki" and tap Done
Now in Safari: Share → Send to Wiki. The URL auto-fills in the app!
